# ‚úÖ **1. Ingress / Input Functions & Suitable DSA**

## **1.1 Polling SFTP/FS/Blob**

**Function:** list files incrementally
**DSA:**

* *Cursor/offset pointer* ‚Üí (stored in DB)
* *Sorted list / priority queue* ‚Üí for ordering by timestamp
* *HashSet* ‚Üí seen files for dedupe

## **1.2 File finalization check**

**Function:** ensure file fully uploaded
**DSA:**

* *Time-series comparison* of file size
* *Sliding window* ‚Üí to ensure stable file size

## **1.3 Idempotent ingestion**

**Function:** no duplicate files
**DSA:**

* *HashMap / HashSet* ‚Üí dedupe store (dedupe_key hash)
* *Bloom Filter (optional)* ‚Üí memory-efficient duplicate checks

## **1.4 Input retries / backoff**

**Function:** recover from transient input failures
**DSA:**

* *Exponential backoff algorithm*
* *Retry queue*
